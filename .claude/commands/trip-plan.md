## AI 여행 계획 도우미

기존 여행 페이지에 AI가 조사한 일정, 맛집, 교통, 예산 정보를 채워넣습니다.

### 사용법
```
/trip-plan {여행폴더명}
```
예: `/trip-plan 2026-tokyo`

### 실행 절차

1. `trips/{여행폴더명}/index.html` 읽기
2. 현재 채워진 정보와 빈 섹션 파악

3. 웹 검색으로 목적지 정보 조사:
   - 추천 관광지 & 동선
   - 현지 맛집 (평점, 가격대, 휴무일)
   - 교통편 (노선, 요금, IC카드, 패스 비교)
   - 예상 날씨 & 옷차림
   - 예산 가이드
   - 사전 준비 사항 (비자, 입국 서류, 환전 등)

4. 사용자에게 조사 결과 요약 제시 → 확인 후 HTML 반영

5. 각 Day의 Leaflet mapData에 좌표 추가

### 조사 시 참고
- 식당은 Google Maps 평점 4.0 이상 위주
- 교통은 IC카드 vs 패스 비교 분석
- 예산은 알뜰/적당/넉넉 3단계로

### 주의사항
- 조사 결과를 **반드시 사용자에게 보여주고 확인 후** HTML에 반영
- 기존에 채워진 데이터는 덮어쓰지 않음 (추가만)
- 실제 예시 참조: `trips/2025-osaka/index.html`
